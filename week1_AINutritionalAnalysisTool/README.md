# 作品名
AIフルオート栄養管理ツール

# 概要
Google AI Studioというサイトから自分のAPIキーを呼び出し、食材を入力するだけで、AIが自動的に食材の栄養価を分析し、PFCバランスを計算してくれるツールです。

# 何を参考にしたのか
マクロファクターという既存の食事管理アプリを参考にしました。

#　動かし方
- https://aistudio.google.com/からAPIキーを取得
- コードを実行し画面上にあるAPIキーに取得したAPIキーを入力かコピペをして貼り付けます。
- 食材を入力して送信すると、あとはAIが自動で食品のPFC値を分析してくれます。

# 生成AIの使い方
自分が作りたい、こんなのあったらいいなと思うアイデアを渡して、コードの書き方や表示するUIを相談しました。

# 学び
- APIキーの取得まではよかったが使えるAIモデルに制限があり、上限もある。またAIモデルが重いのかレスポンスが遅く、処理されないことが多々あって簡単にAIは使えないんだなと感じました。
- なにが作りたいのか、どんな機能がほしいかを明確にして言語化するのが難しい。
