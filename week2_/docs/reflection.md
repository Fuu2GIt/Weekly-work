# 何を自分で理解できたか
- Pythonのリストの基本的な操作方法を理解できた。
- リストの要素の追加、削除を学んだ。
- 要素番号の固定化の面倒さを学んだ
  
# 実装して難しかった点
- リストの要素をdel,removeなどで削除すると要素番号が詰められてしまうので、要素番号を固定化するのが難しかった。

# 足りないと感じた点
- 素のコーディング力
- 知識
  
# 変更によってなにが良くなったか
- 変更前は削除機能がなく追加しかできなったが、削除できるようになった
- 要素番号が詰められてしまうため、番号が変わってしまうが固定化により番号が変わらなくなった

# うまくいかなかった点
- 要素番号の固定化がうまくいかなかった
- 削除機能の実装自体はできたが、詰められ問題がなかなか解決できなかった
