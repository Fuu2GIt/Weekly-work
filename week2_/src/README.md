# 作品名
ターミナルで動く簡易日記帳

# 概要
ターミナル上で動く簡単な日記帳アプリケーションです。ユーザーは日記を追加、表示、削除することができます。

# なにを参考にしたか
noteというサイトを参考にしました。
コードもいくつかのサイトを参考に改変を行いました。

# どんな改変を加えたか
元のコードに日記の削除機能を追加しました。
削除は全記事を削除と記事番号を選択し削除するの2種類を追加。
また削除した際に記事番号が詰められずに番号が固定化されるようにしました。

# 動かし方
実行をするとターミナルで起動、番号ごとの説明が表示されるのでそれに従って操作してください。
日記の追加は1、全記事の表示は2、終了は3、全記事の削除は4、記事番号を選択して削除は5を入力してください。

# 生成AIの使い方
生成AIは最初のコードの生成、及び解説に使いました。

# 学びや気づき
リスト構造、要素削除、要素番号の固定化
調べれば以外と情報がでてきて調べながらコードをつくっていくのが楽しかったこと
